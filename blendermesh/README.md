# BlenderMesh

A Minetest mod for placing schematic models exported from Blender

## Usage
- Place your exported `.lua` schematic files in the `BlenderMesh/models/` folder.
- In-game, use `/placemodel filename.lua` to place the model at your position.

## Exporting from Blender
- Use the Blender exporter to create `.lua` schematic files.
